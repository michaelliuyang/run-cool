Prepare Environment

  1. Application Server
    • Tomcat 6.x or 7.x OR other application servers
  2. DB Server
    • Mysql 5.x OR other DB
  3. JDBC Connection Pool
    • The project JDBC data source used JNDI config.
    • Proxool OR DBCP OR other ...
  4. Install Maven
    • Maven Home

Configuration

Open command or go to RUN-COOL download code zip.

$ git clone https://github.com/michaelliuyang/run-cool.git
$ cd run-cool/run-cool-app
$ mvn install

Get war file from project target directory and deploy your application server.
